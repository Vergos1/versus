# Versus

Agency landing page — portfolio showcase, services and contact section.

![HTML](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)

## About

Versus is a static agency landing page built with pure HTML and CSS. Features a portfolio gallery, hero section with a call-to-action and a responsive layout with custom CSS grid. No frameworks or build tools — vanilla HTML/CSS only.

🔗 **Live:** [versus](https://vergos1.github.io/versus/)

## Pages

| Page | Description |
|---|---|
| **Home** | Hero section with slogan and CTA |
| **Portfolio** | Latest work gallery grid |
| **Contact** | Footer with brand info |

## Features

- **Portfolio gallery** — 3-column image grid with responsive layout
- **Responsive** — custom CSS grid adapts for mobile, tablet and desktop
- **Google Fonts** — Lato font family
- **Hover effects** — nav links and CTA button with color transitions

## Tech Stack

| Technology | Purpose |
|---|---|
| HTML5 | Page structure |
| CSS3 | Styling and layout |
| Google Fonts | Typography |

## Getting Started

No build step required — open directly in browser.

```bash
# Clone the repository
git clone https://github.com/Vergos1/versus.git

# Open in browser
open index.html
```
